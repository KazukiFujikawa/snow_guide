## サイト概要
### サイトテーマ
スノーボーダーがオススメする全国のゲレンデのレビューサイト
​
### テーマを選んだ理由
<!-- - ですます調で記載しましょう。READMEファイルは企業様も見られます。 -->
スノーボードが好きで、シーズンになるとゲレンデに行くことが多いのですが、同じ場所に行くことが増えてきていました。
友人以外のスノーボーダーがどこのゲレンデをオススメしているのかを知れるサービスがあれば便利だと考えました。
経験者だけでなく、初心者の方でも滑りに行きやすいゲレンデを探す助けにもなると考え、このテーマにしました。
​
### ターゲットユーザ
情報を見てすぐにでも行きたい人
週末に車を使って移動することができる社会人
初心者と経験者がいるグループでゲレンデを探している人
いつもとは違うゲレンデを探している人
​
### 主な利用シーン
混雑を避けじっくり練習ができるゲレンデを探したい時
上級者、中級者、初心者向けなのかを知りたい時
広いのか狭めなコースが多いのか知りたい時
ナイターでも滑りやすいのかを確認したい時
